About this page:

This is a collection of calculators for Old School RuneScape.
Currently this page is capable of performing calculations related to melee max hit, accuracy, defence, and DPS.

If you'd like to see how these calculations work, check out this wiki article on the OSRS Wiki.

Potential future additions include mage and range calculators in the same format at these, support for special attacks, additional support for niche cases such as Dharok's set effects, and a database of gear to be selected from.
